# SPlayer 快速开始指南

## 5 分钟快速开始

### 步骤 1: 检查环境（30秒）

```powershell
cd src\BuildScript
.\fix-build-issues.ps1
```

### 步骤 2: 安装 Visual Studio（如需要）

如果脚本显示 "No Visual Studio installation found":

```powershell
cd ..\..
.\install-visual-studio.ps1
```

然后：
1. 按照提示打开 Visual Studio Installer
2. 选择 "Desktop development with C++"
3. 确保选中 "MFC and ATL support"
4. 点击安装

### 步骤 3: 构建项目（5-30分钟）

```batch
cd src\BuildScript
build-fixed.cmd
```

## 详细步骤

### 如果 Visual Studio 已安装

1. **修复构建问题**:
   ```powershell
   cd src\BuildScript
   .\fix-build-issues.ps1
   ```

2. **构建项目**:
   ```batch
   .\build-fixed.cmd
   ```

### 如果 Visual Studio 未安装

1. **运行安装助手**:
   ```powershell
   .\install-visual-studio.ps1
   ```

2. **安装 Visual Studio**:
   - 打开 Visual Studio Installer
   - 选择 "Desktop development with C++"
   - 安装

3. **重启计算机**（如提示）

4. **验证安装**:
   ```powershell
   cd src\BuildScript
   .\fix-build-issues.ps1
   ```

5. **构建项目**:
   ```batch
   .\build-fixed.cmd
   ```

## 预期结果

构建成功后，可执行文件位于：
```
src\out\bin\Release Unicode\splayer.exe
```
